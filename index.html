DPRun

the project can run for design pattern

这是一个实现设计模式的项目，包括了GOF的23中设计模式，使用C#用最常见的语法实现，之所以选择使用C#，是因为我所适用的C#常见语法，这样看起来非常像C++和Java的语法。对于普通程序员阅读没有任何困难。 项目中还包括了一个uml class diagram的项目，其中包括了项目中设计模式的uml类图，如果你对于OOP的理解达到了不用看代码即可明白的程度，那么只需要看此项目中的类图即可。但是如果你必须通过查看代码才能弄明白，那么这个代码没有任何多余的复杂情况，都是基本的框架。 设计模式不需要去了解代码实现细节，需要懂的就是了解骨架和需要解决的问题。 当然设计模式的实现代码也可以让你更深入的理解OOP的内涵。
